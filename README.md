# LTTR/INK for Glyphs

[LTTR/INK](https://www.letterinkapp.com/) gives type designers the handwriting simplicity when the stroke matters. Meet skeleton type design directly in the favourite software Glyphs.

This *master* branch contains LTTR/INK for Glyphs 2. Switch to the *glyphs3* branch for LTTR/INK for Glyphs 3.

### Usage

See our [tutorials](https://www.letterinkapp.com/learn/learn-letterink) to learn how to master the skeletons with LTTR/INK.

### Installation

1. Open *Window > Plugin Manager*
2. Find *LTTR/INK* and click the *Install* button
3. Restart Glyphs.app

After installation, LTTR/INK will appear as *L* button in the app toolbar on the top of the app.
